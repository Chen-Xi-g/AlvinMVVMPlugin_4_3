# Alvin Plugin

## 说明

<!-- Plugin description -->
Alvin Android 快速开发插件，用于一键生成MVVM文件。
<!-- Plugin description end -->

目前暂不支持Java代码, 由于时间原因, 后续会扩展支持.

## 安装

- 手动:
  下载完成后按照以下步骤进行安装
  <kbd>Settings/Preferences</kbd> > <kbd>Plugins</kbd> > <kbd>⚙️</kbd> > <kbd>Install plugin from
  disk...</kbd>
